#### 📚 Libraries

Focused infrastructure libraries for networking, protocol systems, timers, and state machines.

* [go-xtables](https://github.com/singchia/go-xtables): Go bindings for iptables, ebtables, and arptables.
* [geminio](https://github.com/singchia/geminio): One connection. Bidirectional RPC, acked messaging, and stream multiplexing — behind a single net.Conn.
* [geminio-rs](https://github.com/singchia/geminio-rs): Rust port of geminio with Go-compatible wire protocol, async RPC, acknowledged messaging, and stream multiplexing on Tokio.
* [go-timer](https://github.com/singchia/go-timer): Fast Go timer with minimal goroutines.
* [yafsm](https://github.com/singchia/yafsm): Small finite-state machine library for Go.
